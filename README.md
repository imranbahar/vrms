# 🚗 Vehicle Rental Management System (VRMS)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Spring Boot](https://img.shields.io/badge/Spring_Boot-6DB33F?style=for-the-badge&logo=spring-boot&logoColor=white)](https://spring.io/)

A robust, full-stack web application built to streamline vehicle rentals, featuring real-time location searching, secure payment processing, and comprehensive booking management.

## 🚀 Key Features

- **Geospatial Search:** Find available vehicles near your specific location using Geoapify integration.
- **Secure Bookings:** Manage rental periods with real-time validation.
- **Integrated Payments:** Seamless checkout process powered by toyyibPay.
- **Role-Based UI:** Intuitive interface for users to browse and book vehicles effortlessly.

## 🛠️ Built With

- **Backend:** Java, Spring Boot, Spring Security
- **Frontend:** Thymeleaf, Bootstrap 4, CSS/JS
- **Database:** MySQL
- **APIs:** Geoapify (Geocoding), toyyibPay (Payment Gateway)
